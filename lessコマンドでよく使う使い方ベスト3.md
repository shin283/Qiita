## はじめに
ログ調査する時にlessコマンドを使っています。
自分がよく使う使い方のベスト３をまとめます。


## よく使う使い方・オプション
```shell
# 1. 特定の文言で検索して、ファイル内容を確認したいとき
grep 検索したい文言 ファイル名 | less

# 2. ログの１文が長い時に折り返さないで表示したいとき
less -S ファイル名

# 3. 行番号を表示したいとき
less -N ファイル名
```

## lessで表示中のキー操作
| キー |説明 |
|:--:|:--:|
| / | 検索する |
| n | 次を検索する |
| F | 末尾に追加があった場合更新する |
